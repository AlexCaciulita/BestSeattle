# BestInSeattle — Competitor Map (Seattle + PNW)

Last updated: 2026-03-07

## 1) Direct competitors (events/discovery)

1. **EverOut Seattle**
- Strength: large event inventory, established brand, newsletter presence
- Weakness: busy UX, less premium editorial cohesion
- Threat level: High

2. **Do206**
- Strength: event breadth, strong local awareness
- Weakness: listing-heavy, less curated editorial positioning
- Threat level: High

3. **Seattle Met (Things to Do + Best Restaurants)**
- Strength: trusted editorial authority
- Weakness: publication-driven UX vs utility product UX
- Threat level: High

4. **Visit Seattle (Events)**
- Strength: tourism authority + broad event coverage
- Weakness: tourism-first, less neighborhood granularity for locals
- Threat level: Medium-High

5. **The Ticket (Seattle Times)**
- Strength: distribution + media parent
- Weakness: legacy UX patterns
- Threat level: Medium

## 2) Indirect competitors
- Eventbrite / Meetup / Ticketmaster discovery pages
- Instagram/TikTok local creators
- Reddit local recommendation threads
- Google Maps + Yelp list behavior

Threat level: Medium (fragmented attention, not cohesive curation product)

## 3) Strategic gap we can own
1. **Premium clean UI + trust-first curation**
2. **Zone-level precision** (neighborhood + Eastside + PNW edge)
3. **Editorial + utility hybrid** (highly scannable daily use)
4. **Sponsored but tasteful** (clear labels, strict partner quality)

## 4) Differentiation checklist (must-have)
- Weekly signature newsletter with a consistent voice
- "This Week’s Best" short-list format
- Curator notes on why an item is worth it
- Zone cards + quick filters
- Restaurant/place scoring transparency
